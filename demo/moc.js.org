/*
「御品書だよ <summary> は」

効能：
HTML の <details> は，閉じておけば，
そのままで目次としても使えて便利です．
ただし，<summary> 部分をクリックしないと開閉できないのが困り物でした．
・本文が長い場合，閉じるためにページのスクロールも必要．
・リンク経由で <details> に来た場合，目次のどの項目を開くべきか不明．
本ライブラリは，これらの問題点を解決します．

HTMLサンプル断片：
<head>
<script src="moc.js"></script>
</head>
<body onload="Init('<button>Close</button>');">
<div>
<button onclick="OpenAll();">Open All</button>
<button onclick="CloseAll();">Close All</button>
</div>
<details id="A">
<summary>test A</summary>
<a href="#B" onclick="Open(B);">Open and Goto #B.</a>
<a href="#B">Just Goto #B.</a>
</details>
<details id="B">
<summary>test B</summary>
ここは details id="B" です．
<a href="/test/details.html#A" onclick="Open(A);">Open and Goto #A.</a>
<a href="#A">Just Goto #A.</a>
</details>

補足：
同ページへのリンクの場合，再読込が発生しないので，
onclick="Open(...);" の指定も必要．
別ページへのリンクの場合，不要．
*/
